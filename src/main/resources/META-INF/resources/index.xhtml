Ultima

A highly customizable premium application template featuring 4 menu modes, 12 themes, light and dark menu and 2 user profile modes. In total there are 384 possible layout combinations.
Based on material design language, it is fully responsive, touch optimized, built with SASS, CSS3 and HTML5.
Live Demo Docs Support Buy Now

Features

Fully Responsive

Ultima is crafted to provide optimal viewing and
interaction experience for a wide range of devices.

Material Design

Ultima is inspired by Google’s Material Design language.

Full SaSS Support

Scss syntax of Sass is used to create the css of
both layout and theme for easy customization.

12 Different Themes

Ultima provides 12 themes out of the box and
it is also very easy to create your own theme with sass variables.
Here is a video tutorial to show how it can be done in a few minutes.

4 Menu Orientations, Light and Dark Menu, 2 User Profiles

Ultima is highly customizable, in total there are 384 layout combinations.
Menu has 4 modes; Static, Overlay, Horizontal and Slim with Light and Dark color alternatives.
In addition user profile menu can be placed inside the main menu or topbar.

Mobile Experience

Enhanced mobile experience with theming native browser elements and support for special events such as swipe.

Template Pages

Landing, login and error pages are provided
as template pages to get started with building your app in no time.

Well Supported

Ultima has a dedicated forum where lifetime support is delivered by engineers at PrimeTek in a timely manner.
